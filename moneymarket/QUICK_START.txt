- Run on http://localhost:8082

================================================================================
  OPTION 2: QUICK START WITH H2 (NO MYSQL NEEDED)
================================================================================

If MySQL is not running or you want to test without it:

    cd C:\new_cbs3\cbs3\moneymarket
    start-app-h2.bat

This will:
- Use H2 in-memory database (no MySQL required)
- Create application-h2.properties automatically
- Run on http://localhost:8082
- Access H2 Console at http://localhost:8082/h2-console

  TROUBLESHOOTING
================================================================================

ERROR: "Port 8082 is already in use"
FIX: 
    netstat -ano | findstr :8082
    taskkill /PID <process_id> /F

ERROR: "Cannot connect to database"
FIX (Option 1 - Start MySQL):
    net start MySQL80

FIX (Option 2 - Use H2 instead):
    cd C:\new_cbs3\cbs3\moneymarket
    start-app-h2.bat

ERROR: "Maven not found"
FIX: Install Maven or use IDE (Option 5)

ERROR: "Java not found"
